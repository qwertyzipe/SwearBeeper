# Swear Beeper 🔇

Пикает маты в (почти) реальном времени

Приложение, которое слушает микрофон, распознаёт мат в реальном времени и автоматически запикивает его — с небольшой задержкой (~1-2 сек). Полезно для стримов, записи подкастов, звонков в Discord и т.д.

**Этот репозиторий содержит открытый исходный код.** Собранное приложение (`.exe`) публикуется отдельно в [Releases](https://github.com/qwertyzipe/SwearBeeper/releases) — здесь можно посмотреть код, убедиться, что внутри нет ничего вредоносного, и при желании собрать exe самостоятельно из исходников.

## Как это работает

```
Микрофон → кольцевой буфер (полное качество, 48кГц)
         → параллельно: копия звука пересчитывается в 16кГц и уходит в Vosk (офлайн-распознавание речи)
         → если распознанное слово похоже на мат → на буфер накладывается бип (или случайный кастомный звук)
         → очищенный звук (с задержкой ~1-2 сек) уходит на выход (динамики / виртуальный микрофон)
```

Внутри задержки хватает времени, чтобы: услышать слово → распознать его (в том числе по промежуточным, ещё не завершённым результатам Vosk) → решить, мат это или нет → наложить бип **до того**, как этот кусок аудио уйдёт на выход.

Распознавание речи полностью **офлайн** (через [Vosk](https://alphacephei.com/vosk/)) — никакой звук никуда не отправляется в интернет.

## Возможности

**Детект и цензура**
- Определение мата по словарю корней + типичных приставок
- Детект по словам в реальном времени, а не только по завершённым фразам
- Белый список слов-исключений
- Свой список запрещённых слов — добавление/удаление/импорт/экспорт прямо в интерфейсе
- Настраиваемая задержка, громкость бипа, паддинг до/после слова
- Кастомные звуки вместо бипа — можно загрузить несколько .wav-файлов, при мате выбирается случайный

**Аудио**
- Захват и воспроизведение в полном качестве (48кГц)
- Регулируемое усиление микрофона (gain)
- Выбор микрофона и устройства вывода
- Тест микрофона

**Интерфейс и удобство**
- Системный трей
- Глобальный хоткей мьют/анмьют
- Измеритель уровня звука
- Автосохранение всех настроек
- Кнопки сброса (↺) у каждого слайдера
- Подсказки у настроек

**Статистика и журнал**
- Счётчик матов за сессию и за всё время
- Таблица-рейтинг слов по частоте
- Хронологический журнал матов с датой/временем

**Интеграция с OBS**
- OBS-скрипт (obs_script/swear_beeper_obs.py) подключается к приложению по локальному сокету
- Текстовый счётчик матов на сцене OBS
- Баннер "ЦЕНЗУРА" при каждом мате
- Звук при цензуре
- Таймер "без мата уже: ЧЧ:ММ:СС"

## Ограничения

- Это эвристика, а не идеальный ИИ-классификатор — возможны ложные срабатывания и пропуски
- Маленькая модель Vosk (~50 МБ) может ошибаться в шумной обстановке
- Без задержки вывода технически невозможно "предсказать" мат до того, как он прозвучит
- Нет шумоподавления/эхоподавления — осознанное решение, чтобы не терять четкость распознавания
- Нужен нормальный микрофон

## Требования

- Windows (тестировалось на 10/11)
- Python 3.12 (для запуска из исходников)
- Модель Vosk для русского языка

## Установка зависимостей

```bash
py -3.12 -m pip install sounddevice vosk numpy pystray pillow keyboard
```

## Модель Vosk

Скачай отдельно: https://alphacephei.com/vosk/models
- Для старта: vosk-model-small-ru-0.22 (~50 МБ)
- Для точности: vosk-model-ru-0.42 (~1.8 ГБ, но медленнее — не рекомендую для живого запикивания)

Распакуй в папку `model_ru` рядом со скриптом.

## Запуск из исходников

```bash
py -3.12 beeper_gui.py
```

## Использование с Discord / OBS / другими программами

1. Установи VB-CABLE: https://vb-audio.com/Cable/ (желательно перезагрузить)
2. В приложении выбери выход **CABLE Input**
3. В Discord/OBS/(другое) выбери микрофон **CABLE Output**
4. Выключи шумодав
5. Проверь через "Тест микрофона"

## Интеграция с OBS

Смотри obs_script/README_OBS.md. Приложение открывает порт 47823, OBS-скрипт подключается и обновляет счётчик/баннер/таймер.

Важно: OBS поддерживает Python 3.6-3.10 официально, часто реально работает до 3.10-3.12.

# Сборка exe

```bash
py -3.12 -m pip install pyinstaller
py -3.12 -m PyInstaller --onefile --windowed --name SwearBeeper --icon=icon.ico --add-data "icon.ico;." --add-data "model_ru;model_ru" --collect-all vosk --collect-all pystray --collect-all keyboard --clean beeper_gui.py
```

Важно: всегда `py -3.12 -m PyInstaller`, не голое `pyinstaller`.

## Проблемы при сборке

- **"pyinstaller is not recognized"**: используй `py -3.12 -m PyInstaller`
- **Ошибка про vosk/pystray/keyboard в exe**: проверь флаги `--collect-all`, пересобери с `--clean`
- **PermissionError при сборке**: закрой старый exe полностью (проверь трей — приложение может тихо работать в фоне)
- **Антивирус ругается**: ложная тревога для PyInstaller `--onefile`, читай исходники
- **"Папка модели не найдена"**: нажми "Сбросить путь" рядом с полем модели
- **Иконка не появляется в панели задач/трее**: убедись, что добавлен флаг `--add-data "icon.ico;."` отдельно от `--icon=icon.ico`; также Windows агрессивно кэширует иконки exe — если не помогло, попробуй переименовать exe или почистить кэш иконок системы
